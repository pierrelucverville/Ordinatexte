# Ordinatexte

Ordinatexte est un générateur automatique de texte philosophico-poétique en français,
fondé sur un moteur syntaxique déterministe (Patriarche),
sans apprentissage automatique ni intelligence artificielle.

## Contenu du dépôt

- `index.html` : interface minimale de génération
- `patriarche.js` : moteur de génération syntaxique

## Utilisation

Ouvrir `index.html` dans un navigateur moderne.
Cliquer sur « Générer ».
